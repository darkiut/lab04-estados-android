## Descripción
Proyecto completo sobre manejo de estados en Jetpack Compose con Git avanzado.

## Contenido del Laboratorio

### Parte 0: Git Avanzado (3 puntos)
- ✅ Múltiples commits
- ✅ Git revert
- ✅ Ramas y merge
- ✅ Resolución de conflictos

### Partes 1-3: MovieCounter (3 puntos)
- ✅ Contador estático
- ✅ Estado con `remember`
- ✅ Estado persistente con `rememberSaveable`

### Parte 4: Task Manager (6 puntos)
- ✅ Aplicación completa de gestión de tareas
- ✅ Estados persistentes
- ✅ CRUD completo
- ✅ Estadísticas en tiempo real

### Parte 5: Estados Avanzados (Tutorial)
- ✅ State Hoisting
- ✅ Derived State
- ✅ Side Effects
